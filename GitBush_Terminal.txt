						КОМАНДЫ В ТЕРМИНАЛЕ

1.	cd  – выйти в корень папки
2.	ls –la   -вывести список содержимого папки, где нахожусь (для MAC)
3.	dir  - -вывести список содержимого папки, где нахожусь (для винды)
4.	cd  название папки  -зайти в папку
5.	cd ..   –выйти из папки
6.	mkdir  название папки   -создать папку
7.	mkdir  dir_1 dir_2 dir_3   - создать несколько папок
8.	mkdir {00..10}  -создать несколько папок (если их много и их всех не прописывать)
9.	touch  название файла.формат   -создать файл в формате (на винде)
10.	> название файла.формат    - создать файл в формате  (команда на MAC)
11.	touch 1.txt 2.txt 3.txt    -создать несколько файлов
12.	vim название файла.формат  -зайти в режим редактирования в файл и там можно писать
13.	esc :wq   -выйти из vim
14.	open ./   -открывается просмотрщик файлов  (команда на MAC)
15.	start .   –на винде открывается просмотрщик файлов (команда для винды)
16.	mv название файла.формат  (куда перемещаем->)название папки/ название файла.формат    -переместить файл в другую папку; или прописываем полностью пути (текущее расположение файла с названием и форматов –куда переместить полный путь с названием этого же файла.формат; или находясь в папке на порядок выше указываем ./название папки/файл.формат)
17.	pwd   -показать путь к текущему файлу
18.	rm  -r название папки   -удалить папку
19.	rm название файла.формат  - удалить файл
20.	cp  название.формат путь/название.формат    –копировать файл
21.	find . –name название.формат    - поиск файла ниже / в файловой системе
22.	cat – посмотреть содержимое файла
23.	less  - посмотреть содержимое с прокруткой (для длинных файлов)
24.	grep ‘слово’  название.формат   - вывести все строки, где встречается слово в ковычках
25.	date   -вывести дату и время
